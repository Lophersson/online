Casos de Prueba con Validaciones y Errores

10 funciones, cada una con 4 pruebas incluyendo errores o respuestas no esperadas

Caso 1: dividir(a, b)

Input: (10, 2) → Esperado: 5
Input: (8, 0) → Error: División por cero
Input: (9, 3) → Esperado: 3
Input: (12, "x") → Error: Tipo de dato inválido

Caso 2: obtenerLetraDNI(numeroDNI)

Input: (12345678) → Esperado: Z
Input: (-1) → Error: Número de DNI no válido
Input: ("ABC") → Error: DNI debe ser numérico
Input: (98765432) → Esperado: M

Caso 3: calcularAreaRectangulo(largo, ancho)

Input: (5, 3) → Esperado: 15
Input: (-2, 4) → Error: Dimensiones no pueden ser negativas
Input: (7, 2) → Esperado: 14
Input: (null, 5) → Error: Valor no definido

Caso 4: registrarUsuario(nombre, correo, contraseña)

Input: ("Luis", "luis@mail.com", "abc123") → Esperado: Registro exitoso
Input: ("", "mail@mail.com", "clave") → Error: Nombre vacío
Input: ("Ana", "correoSinArroba", "clave") → Error: Correo inválido
Input: ("Marta", "marta@gmail.com", "clave456") → Esperado: Registro exitoso

Caso 5: calcularIMC(peso, altura)

Input: (70, 1.75) → Esperado: 22.86
Input: (65, 0) → Error: División por cero
Input: (50, 1.6) → Esperado: 19.53
Input: ("peso", 1.7) → Error: Peso inválido

Caso 6: buscarProducto(lista, producto)

Input: (["pan", "leche"], "pan") → Esperado: Encontrado
Input: (["queso", "huevo"], "jamón") → Esperado: No encontrado
Input: ([], "leche") → Error: Lista vacía
Input: (null, "pan") → Error: Lista no válida

Caso 7: convertirCelsiusAFahrenheit(celsius)

Input: (0) → Esperado: 32
Input: (100) → Esperado: 212
Input: ("treinta") → Error: Entrada no numérica
Input: (-40) → Esperado: -40

Caso 8: concatenarNombres(nombre1, nombre2)

Input: ("Juan", "Pérez") → Esperado: Juan Pérez
Input: (123, "García") → Error: Nombre inválido
Input: ("Ana", "López") → Esperado: Ana López
Input: ("", "") → Error: Ambos campos vacíos

Caso 9: esFechaValida(día, mes, año)

Input: (31, 12, 2020) → Esperado: Fecha válida
Input: (31, 2, 2021) → Error: Fecha inválida
Input: (29, 2, 2024) → Esperado: Fecha válida (año bisiesto)
Input: ("uno", 3, 2022) → Error: Día no numérico

Caso 10: generarClaveSegura(longitud, incluirSimbolos)

Input: (8, true) → Esperado: Clave generada con símbolos
Input: (12, false) → Esperado: Clave alfanumérica
Input: (3, true) → Error: Longitud insuficiente
Input: ("diez", false) → Error: Longitud no numérica
