Función de Repetición con For Anidados

Código:

static string RepetirTexto(string texto, int x)
{
    string resultado = "";

    for (int i = 0; i < texto.Length; i++)
    {
        for (int j = 0; j < x; j++)
        {
            resultado = resultado + texto[i];
        }
    }

    return resultado;
}

Funcionamiento:

Esta función recibe una cadena texto y un número entero x. Por cada carácter del texto:

  1. El primer bucle i recorre cada letra del texto.
  2. El segundo bucle j repite cada letra x veces.
  3. Se va concatenando cada letra x veces al string resultado.

Caso de prueba:

Entrada: texto = "Hola", x = 3

Proceso:
Letra 'H' → repetir 3 veces → "HHH"
Letra 'o' → repetir 3 veces → "OOO"
Letra 'l' → repetir 3 veces → "LLL"
Letra 'a' → repetir 3 veces → "AAA"

Resultado final: HHHOOOLLLAAA
