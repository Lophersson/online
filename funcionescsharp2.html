Simulación de Funciones estilo C# con múltiples entradas

1. Contar vocales (string texto)

return texto.Count(c => "aeiouAEIOU".Contains(c));
EntradaResultado
"Hola mundo"4
"CSharp"1
"AEIOUaeiou"10
""0

2. Invertir texto (string texto)

char[] arr = texto.ToCharArray(); Array.Reverse(arr); return new string(arr);
EntradaResultado
"casa""asac"
"12345""54321"
"Hola!""!aloH"
""""

3. Validar email (string email)

if (email.Contains("@") && email.Contains(".")) return "Válido"; else return "Inválido";
EntradaResultado
"test@mail.com"Válido
"user@dominio"Inválido
"@mail.com"Válido
"texto plano"Inválido

4. Contar palabras (string texto)

return texto.Split(' ').Length;
EntradaResultado
"Hola mundo"2
"Uno dos tres cuatro"4
"Solo"1
""1

5. Substring seguro (string texto, int inicio, int longitud)

if (inicio+longitud > texto.Length) return "Error"; else return texto.Substring(inicio, longitud);
EntradaResultado
"Programación", 0, 4"Prog"
"Hola", 1, 2"ol"
"Texto", 3, 5Error
"ABCDEF", 2, 2"CD"

6. Verificar número positivo (int n)

return n >= 0 ? "Positivo" : "Negativo";
EntradaResultado
5Positivo
0Positivo
-10Negativo
99Positivo

7. Comparar cadenas (string a, string b)

return a == b ? "Iguales" : "Distintas";
EntradaResultado

"Hola", "hola"Distintas
"123", "123"Iguales
"uno", "dos"Distintas

8. Formatear nombre (string nombre, string apellido)

return apellido + ", " + nombre;

"Ana", "García""García, Ana"
