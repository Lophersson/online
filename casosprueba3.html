Casos de Prueba Complejos

10 funciones con múltiples parámetros y 4 pruebas por función

Caso 1: calcularPromedio(n1, n2, n3)

Input: (5, 10, 15) → Esperado: 10
Input: (0, 0, 0) → Esperado: 0
Input: (20, 30, 40) → Esperado: 30
Input: (-5, 5, 10) → Esperado: 3.33

Caso 2: esTrianguloValido(a, b, c)

Input: (3, 4, 5) → Esperado: true
Input: (1, 2, 3) → Esperado: false
Input: (5, 5, 5) → Esperado: true
Input: (0, 4, 5) → Esperado: false

Caso 3: validarRango(min, max, valor)

Input: (1, 10, 5) → Esperado: true
Input: (1, 10, 0) → Esperado: false
Input: (10, 100, 100) → Esperado: true
Input: (20, 30, 31) → Esperado: false

Caso 4: sumaProductos(a, b, c)

Input: (2, 3, 4) → Esperado: 2×3 + 3×4 = 6 + 12 = 18
Input: (1, 2, 3) → Esperado: 1×2 + 2×3 = 2 + 6 = 8
Input: (0, 5, 7) → Esperado: 0 + 35 = 35
Input: (-1, 4, 2) → Esperado: -4 + 8 = 4

Caso 5: calcularDistancia(x1, y1, x2, y2)

Input: (0,0,3,4) → Esperado: 5
Input: (1,1,1,1) → Esperado: 0
Input: (2,-1,4,3) → Esperado: 4.47
Input: (-3,-4,0,0) → Esperado: 5

Caso 6: verificarAcceso(usuario, clave, intento)

Input: ("admin", "1234", "1234") → Esperado: Acceso concedido
Input: ("admin", "1234", "0000") → Esperado: Acceso denegado
Input: ("admin", "", "") → Esperado: Acceso denegado
Input: ("user", "pass", "pass") → Esperado: Acceso concedido

Caso 7: aplicarDescuento(precio, porcentaje)

Input: (100, 10) → Esperado: 90
Input: (200, 25) → Esperado: 150
Input: (50, 0) → Esperado: 50
Input: (80, 100) → Esperado: 0

Caso 8: calcularSalario(base, horasExtra, pagoExtra)

Input: (1000, 5, 20) → Esperado: 1100
Input: (1200, 0, 15) → Esperado: 1200
Input: (900, 10, 10) → Esperado: 1000
Input: (1000, 2, 25) → Esperado: 1050

Caso 9: esDentroDelHorario(horaInicio, horaFin, horaActual)

Input: (9, 17, 10) → Esperado: true
Input: (9, 17, 8) → Esperado: false

Input: (22, 6, 7) → Esperado: false

Caso 10: generarResumen(nombre, edad, ciudad)

Input: ("Luis", 30, "Madrid") → Esperado: "Luis, 30 años, vive en Madrid"
