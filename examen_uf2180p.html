Examen de Arquitectura de Capas en C#

1. ¿Qué capas se comunican con la capa de negocio?

Capa de presentación y capa de datos.

2. ¿Cuáles son las tres capas tradicionales en una arquitectura en C#?

Presentación, Negocio, Datos.

3. ¿Qué capas están directamente relacionadas en una comunicación básica?

Presentación y Negocio.

4. ¿Qué capa se debe modificar al migrar de SQL Server a MySQL?

Capa de datos.

5. ¿Qué sucede si la presentación no accede a negocio y negocio no accede a datos?

El negocio usará clases o funciones propias para acceder a los datos.

6. ¿En qué capa se utiliza la clase SqlConnection?

Capa de datos.

7. ¿Qué representa una variable que instancia SqlConnection?

Una conexión a la base de datos.

8. ¿Qué tipo de error puede producir una lista vacía sin elementos?

Excepción al intentar acceder a elementos inexistentes.

9. ¿Qué ocurre si se realiza una conexión sin cerrar la anterior?

Puede lanzar una excepción capturable en el bloque catch.

10. ¿Dónde se ubica la lógica de consultas SQL?

En la capa de datos.

11. ¿Qué sucede si micodigo no es un número entero?

Se produce un error de conversión.

12. ¿Qué sucede si se intenta borrar un registro que no existe?

No se borra nada.

13. ¿Dónde se encuentra el acceso a la base de datos?

En la capa de datos.

14. ¿Qué sucede si no hay primary key y varios registros tienen el mismo código?

Se borran todos los registros con ese código.

15. ¿Qué se actualiza si se hace un SET nombre en un UPDATE con filtro por código?

El nombre del registro con ese código.

16. ¿Cómo se agregan nombres de una lista a un ListBox en C#?

foreach (var nombre in nombres) jugadorelb.Items.Add(nombre);

17. ¿Cómo funciona un método de borrado que devuelve un booleano?

Devuelve true si se ejecuta correctamente, false si falla.

18. ¿Qué relación tiene la capa presentación con las otras capas?

Llama a la capa de negocio, que a su vez llama a la capa de datos.

19. ¿Qué se modifica si hay una librería intermedia entre negocio y datos?

Solo se modifica la capa de presentación.

20. ¿Qué capas deben modificarse al cambiar de SQL Server a Oracle?

Capa de datos y capa de negocio (no se toca la presentación).
