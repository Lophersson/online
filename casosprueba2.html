Casos de Prueba Generales

20 funciones diferentes, 4 pruebas cada una

Caso 1: esPar(numero)

Input: 2 → Esperado: true

Input: 0 → Esperado: true
Input: -4 → Esperado: true

Caso 2: invertirCadena(cadena)

Input: "hola" → Esperado: "aloh"

Input: "123" → Esperado: "321"
Input: "A b" → Esperado: "b A"
