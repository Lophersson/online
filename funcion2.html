Función FunctionTexto en C#

Código:

static string FunctionTexto(string texto, int x)
{
    string resultado = "";

    for (int i = 0; i < texto.Length; i++)
    {
        char c = texto[i];
        resultado = resultado + c;

        if (i > x)
        {
            resultado = resultado + c;
        }
    }

    return resultado;
}

Funcionamiento:

Esta función toma un texto y un número entero x, y construye un nuevo string aplicando esta lógica:

  1. Recorre el texto carácter por carácter.
  2. Siempre añade el carácter actual al resultado.
  3. Si el índice actual i es mayor que x, repite el carácter (lo añade otra vez).
  4. Devuelve el string resultante.

Caso de prueba:

Entrada: texto = "Hola", x = 2

Proceso:
i = 0 → 'H' → resultado = "H"
i = 1 → 'o' → resultado = "Ho"
i = 2 → 'l' → resultado = "Hol"
i = 3 → 'a' → como 3 > 2, se duplica → resultado = "Holaa"

Resultado final: Holaa
