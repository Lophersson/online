Funciones C# simuladas: Bucles y Substrings

1. Extraer iniciales (string nombreCompleto)

string[] partes = nombreCompleto.Split(' ');
string ini = "";
foreach (string p in partes) ini += p[0];
return ini.ToUpper();
EntradaResultado
"Juan Carlos Pérez"JCP
"Ana María López"AML
"luis"L
"pedro alonso"PA

2. Substring de longitud fija cada N caracteres

for (int i = 0; i < texto.Length; i += 3)
  segmentos += texto.Substring(i, Math.Min(3, texto.Length - i)) + "-";
EntradaResultado
"ABCDEFGHIJ"ABC-DEF-GHI-J-
"123456789"123-456-789-
"HolaMundo"Hol-aMu-ndo-
"Hi"Hi-

3. Duplicar vocales en una cadena

string resultado = "";
foreach (char c in texto)
  resultado += "aeiouAEIOU".Contains(c) ? $"{c}{c}" : c.ToString();
EntradaResultado
"Hola"Hoolaa
"Mundo"Muundoo
"AEIOU"AAEEIIOOUU
"BCDF"BCDF

4. Contar ocurrencias de un carácter (char objetivo)

int contador = 0;
foreach (char c in texto)
  if (c == objetivo) contador++;
return contador;
EntradaResultado
"banana", 'a'3
"mississippi", 's'4
"HELLO", 'L'2
"xyz", 'z'1
