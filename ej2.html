Explicación de la Función FuncionSumaEdades

Código del Pseudocódigo

        INICIO
            FUNCION int FuncionSumaEdades()
                INICIO
                    DECLARAR RESULT COMO ENTERO = 0
                    DECLARAR CONTADOR COMO ENTERO = 0
                    DECLARAR LONGITUD COMO ENTERO = LONGITUD(ListaCiudades)

                    MIENTRAS CONTADOR < LONGITUD HACER
                        SI (ListaCiudades[CONTADOR] == "Ferrol") ENTONCES
                            RESULT = RESULT + ListaEdades[CONTADOR]
                        FIN_SI
                        CONTADOR = CONTADOR + 1
                    FIN_MIENTRAS

                    RETORNAR RESULT
                FIN
            FIN_FUNCION
        FIN

Explicación

La función FuncionSumaEdades recorre una lista de ciudades y, si la ciudad es "Ferrol", suma las edades correspondientes en la lista ListaEdades. Al final, retorna la suma total de todas las edades asociadas a "Ferrol".

  • La función declara tres variables: RESULT (para almacenar la suma de las edades), CONTADOR (como índice para recorrer las listas), y LONGITUD (que almacena el tamaño de la lista ListaCiudades).
  • En el bucle MIENTRAS, la función verifica si la ciudad en la posición CONTADOR es "Ferrol". Si es así, se suma la edad correspondiente de la lista ListaEdades al valor de RESULT.
  • El bucle continúa hasta que se han procesado todas las ciudades en ListaCiudades.
  • Finalmente, la función retorna el valor de RESULT, que es la suma total de todas las edades asociadas a "Ferrol".

        ListaEdades = [23, 12, 35, 11]

Detalles del Caso de Prueba

Cuando la función recibe las listas de ciudades y edades mencionadas, el proceso sería el siguiente:

  • Inicialmente, RESULT = 0 y CONTADOR = 0.
  • En la primera iteración del bucle, la ciudad es "Madrid" y no se realiza ninguna suma, luego CONTADOR = 1.
  • En la segunda iteración, la ciudad es "Ferrol" y se suma la edad 12 a RESULT, quedando RESULT = 12, luego CONTADOR = 2.
  • En la tercera iteración, la ciudad es "Ferrol" y se suma la edad 35 a RESULT, quedando RESULT = 47, luego CONTADOR = 3.
  • En la cuarta iteración, la ciudad es "Ferrol" y se suma la edad 11 a RESULT, quedando RESULT = 58, luego CONTADOR = 4.
  • El bucle termina porque CONTADOR = 4 ya no es menor que LONGITUD = 4.
  • La función retorna RESULT = 58.

Conclusión

La función FuncionSumaEdades es útil cuando se necesita realizar una suma condicional de valores, en este caso, las edades de personas que viven en la ciudad de "Ferrol". Este tipo de función puede adaptarse a diferentes necesidades donde sea necesario sumar solo los elementos que cumplen una condición específica.
